Introduction

Introduction

sablib is a C++ library for smoothing and baseline estimation. It provides multiple methods that can be chosen based on your specific needs. Since it is written in C++, it is designed to operate relatively fast. It can be used for the analysis of experimental data such as chromatograms and spectroscopic spectra.

sablib is still a work in progress. More algorithms will be added gradually in the future.

Algorithms

Smoothing

  • sablib::MovingAverage(): Simple moving average
  • sablib::WeightedMovingAverage(): Weighted moving average
    • sablib::GaussianKernel() / sablib::GaussianFilter(): Gaussian filter
  • sablib::SavitzkyGolay(): Savitzky-Golay
  • sablib::Whittaker(): Whittaker Smoother
  • sablib::PSpline(): Penalized Spline(P-Spline)

Baseline Estimation

  • sablib::BaselineLinear(): Linear line between two points
  • sablib::BaselinePolynomial(): Polynomial line
  • sablib::BaselineSpline(): Cubic spline
  • sablib::BaselineSMA(): Simple moving average
  • sablib::BaselineModPoly(): ModPoly
  • sablib::BaselineIModPoly(): IModPoly
  • sablib::BaselineAsLS(): AsLS
  • sablib::BaselineAirPLS(): airPLS
  • sablib::BaselineArPLS(): arPLS
  • sablib::BaselineBeads(): BEADS
